# Everithing is craftable

Addon that adds crafting recepies to all uncraftable items and blocks

## Platform

> Bedrock:
> Android, IOS, XBOX, Playstation, Windows 10

> Version
> 1.19.20


### Still in early development

list of now available recepies:

- ores
- saplings
- sands
- gravel
- bedrock
- cobblestone
- chainmail armor
- grass
